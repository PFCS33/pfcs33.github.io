# Selections

D3 selections let you choose some HTML or SVG elements and change their style and/or attributes.

# Making selections

  • d3.select and d3.selectAll
  • take a string as its only argument.
    • specifies which elements to select
    • in the form of a CSS selector string

# Modifying elements

  • all elements in the selection will be modified.
  • you can pass not only constant values but also a function
    • The function accepts two arguments which are usually named d and i.
d3.selectAll("circle").attr("cx", function (d, i) {
  return i * 100;
});
Name Behaviour Example
.style Update the style d3.selectAll('circle').style('fill', 'red')
.attr Update an attribute d3.selectAll('rect').attr('width', 10)
.classed Add/remove a class attribute d3.select('.item').classed('selected', true)
.property Update an element's property d3.selectAll('.checkbox').property('checked', false)
.text Update the text content d3.select('div.title').text('My new book')
.html Change the html content d3.select('.legend').html('<div class="block"></div><div>0 - 10</div>')

# Event handling

  • .on method, add event handlers to selections.
    • need two arguments
      • a string specifying the event type
      • a 'callback function': was passed the datum d and index i
Event name Description
click Element has been clicked
mouseenter Mouse pointer has moved onto the element
mouseover Mouse pointer has moved onto the element or its children
mouseleave Mouse pointer has moved off the element
mouseout Mouse pointer has moved off the element or its children
mousemove Mouse pointer has moved over the element

# Inserting and removing elements

  • .append, .insert, .remove

# .each

  • lets you call a function for each element of a selection.  - The callback function has two arguments usually named d and i

# .call

  • allows a function to be called into which the selection itself is passed as the first argument

# Filter and sort selections

  • .filter method
    • need a filter function
      • this function has two arguments usually named d and i
    • return selections
  • .sort method
    • need a comparator function
    • has two arguments, usually a and b, which represent the datums on the two elements being compared
    • Sorting only really makes sense if data has been joined

# Data joins

  • The general pattern for creating a data join
    • element-type is a string describing the type of element you’re joining (e.g. ‘div' or 'circle')
d3.select(container)
  .selectAll(element - type)
  .data(array)
  .join(element - type);

# Update function

let myData = [40, 10, 20, 60, 30];
function update(data) {
	d3.select('.chart')
	.selectAll('circle')
	.data(data)
	.join('circle')
	.attr('cx', function(d, i) {
		return i * 100;    })
	.attr('cy', 50)
	.attr('r', function(d) {
		return 0.5 * d;    })
	.style('fill', function(d) {
		return d > 30 ? 'orange' : '#eee';    });
	}

	update(myData)
}

# key function

  • u can pass a key function into the .data method.
    • should return a unique id value for each array element

# enter & update & exit

  • ou can treat entering and exiting elements differently by passing functions into the .join method:
    • Each function has a single parameter:
      • the enter function's parameter enter is the enter selection which represents the elements that need to be created
      • the update function's parameter update is a selection containing the elements that are already in existence (and aren't exiting)
      • the exit function's parameter exit is the exit selection and contains elements that need to to be removed
.join(
	function(enter) {    ...  },
	function(update) {    ...  },
	function(exit) {    ...  }
)

# Shapes
line (opens new window) Generates path data for a multi-segment line (typically for line charts)
area (opens new window) Generates path data for an area (typically for stacked line charts and streamgraphs)
stack (opens new window) Generates stack data from multi-series data
arc (opens new window) Generates path data for an arc (typically for pie charts)
pie (opens new window) Generates pie angle data from array of data
symbol (opens new window) Generates path data for symbols such as plus, star, diamond
var lineGenerator = d3.line();
var points = [
  [0, 80],
  [100, 100],
  [200, 30],
  [300, 50],
  [400, 40],
  [500, 80],
];
var pathData = lineGenerator(points); // returns "M0,80L100,100L200,30L300,50L400,40L500,80"
d3.select("path").attr("d", pathData);

var pieGenerator = d3.pie();
var data = [10, 40, 30, 20, 60, 80];
var arcData = pieGenerator(data);

// arcData is an array of objects: [
//   {
//     data: 10,
//     endAngle: 6.28...,
//     index: 5,
//     padAngle: 0,
//     startAngle: 6.02...,
//     value: 10
//   },
//   ...
// ]

var arcGenerator = d3.arc().innerRadius(20).outerRadius(100);

d3.select("g")
  .selectAll("path")
  .data(arcData)
  .enter()
  .append("path")
  .attr("d", arcGenerator);

# Axes

  • Axis generator: d3.axisBottom, d3.axisTop, d3.axisLeft or d3.axisRight
    • need a scale function
      • the scale domain determines the minimum and maximum tick values
      • the range determines the length of the axis
  • To create an axis:
    • make an axis generator function using d3.axisBottom, d3.axisTop, d3.axisLeft or d3.axisRight (and pass in your scale function)
    • select the container element and pass the axis generator into .call
let scale = d3.scaleLinear().domain([0, 100]).range([0, 500]);

let axis = d3.axisBottom(scale);

d3.select('svg g')
  .call(axis);

# D3 Map

  • The 3 concepts that are key to understanding map creation using D3 are:
    • GeoJSON
      • a JSON-based format for specifying geographic data
      • Feature: Each feature consists of geometry and properties.
        • Properties can contain any information about the feature such as name, id, and other data such as population
    {
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "name": "Africa"
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[-6, 36], [33, 30], ... , [-6, 36]]]
      }
    },
    {
      "type": "Feature",
      "properties": {
        "name": "Australia"
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[143, -11], [153, -28], ... , [143, -11]]]
      }
    },
    {
      "type": "Feature",
      "properties": {
        "name": "Timbuktu"
      },
      "geometry": {
        "type": "Point",
        "coordinates": [-3.0026, 16.7666]
      }
    }
  ]
}
    • projections
      • functions that convert from latitude/longitude co-ordinates to x & y co-ordinates
      • D3 provides a large number of projection functions.
    • geographic path generators
      • functions that convert GeoJSON objects into SVG or Canvas paths
      • .geoPath
        • configure it with a projection function

# a basic map

let geoJson = {
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "name": "Africa"
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[-6, 36], [33, 30], ... , [-6, 36]]]
      }
    },
    ...
  ]
}

let projection = d3.geoEquirectangular();

let geoGenerator = d3.geoPath()
  .projection(projection);

// Join the FeatureCollection's features array to path elements
let u = d3.select('#content g.map')
  .selectAll('path')
  .data(geojson.features)
  .join('path')
  .attr('d', geoGenerator);
